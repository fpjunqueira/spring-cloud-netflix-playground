# Hystrix Client

Circuit Breaker with Netflix Hystrix

- Fallbacks to prevents cascading failures.
- Monitoring with the Hystrix Dashboard and Turbine.

**Build**

`$ ./mvnw clean install
`

**RUN**


`$ ./mvnw spring-boot:run
`
