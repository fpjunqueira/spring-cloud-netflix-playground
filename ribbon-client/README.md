# Ribbon Client

Ribbon is a client side load balancer with the following features:

- Automatically integrates with Eureka.
- Built in failure resiliency (Hystrix)
- Caching/Batching
- Multiple protocols (HTTP, TCP, UDP)

**Build**

`$ ./mvnw clean install
`

**RUN**


`$ ./mvnw spring-boot:run
`
